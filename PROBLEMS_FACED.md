## 📊 Challenges Faced and Resolutions

| Challenge | Description | Resolution |
|----------|------------|------------|
| Dynamic Web Project Option Missing | The option to create a Dynamic Web Project was not available in the IDE environment | Installed the Enterprise Java and Web Developer Tools package in Eclipse to enable web development features |
| MySQL JDBC Driver Configuration | Faced difficulty in establishing database connectivity due to missing MySQL JDBC driver | Downloaded the MySQL Connector/J JAR file and added it to the project's build path |
| Incorrect Credential Usage in Database Connection | Initially used application login credentials instead of database credentials in the connection string | Corrected the configuration by using valid MySQL server credentials (username and password) |
